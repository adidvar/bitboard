bitboard::Position class
#include <bitboard/position.hpp>

The Position class describes a chess position on the board.

Contents

  • Reference
    • Public types
    • Public static functions
    • Constructors, destructors, conversion operators

Internally, the position is represented by a single uint8_t index, ranging from 0 to 63, where 0 corresponds to the bottom-left square (A1) and 63 corresponds to the top-right square (H8). An invalid position is represented by the index kPositionInvalid (64).

Explicit constructor from a linear index.

bitboard::Position::Position(int_t x_pos, int_t y_pos) constexpr noexcept

Constructor from x and y coordinates.

Parameters
x_pos The x-coordinate (column) of the position (0-7).
y_pos The y-coordinate (row) of the position (0-7).

Initializes the position with the given x and y coordinates (0-7). The x-coordinate represents the column (A=0, H=7), and the y-coordinate represents the row (1=0, 8=7). If the coordinates are out of the valid range, the position is invalidated.

bitboard::Position::Position(std::string_view string) explicit constexpr

Explicit constructor from a string representation (e.g., "a1", "H8").

Parameters
string The algebraic notation string representing the position.

Initializes the position from a standard algebraic notation string. The string should be two characters long, with the first character being a lowercase letter from 'a' to 'h' (representing the column) and the second character being a digit from '1' to '8' (representing the row). If the string is invalid, the position is invalidated.

auto bitboard::Position::rotate() const noexcept -> Position constexpr noexcept

Rotates the position 180 degrees on the board.

Returns A new Position object representing the rotated position. Returns an invalid Position if the original is invalid.

This operation effectively mirrors the position across the center of the board. An invalid position remains invalid after rotation.
