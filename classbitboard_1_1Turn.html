bitboard::Turn class
#include <bitboard/turn.hpp>

Represents a chess turn (move).

Contents

  • Reference
    • Public static functions
    • Constructors, destructors, conversion operators
    • Public functions

Encapsulates move information: start/end positions, promotion figure (if pawn promotion), and flags for validity, promotion (is a pawn promotion), and triviality (simple move). Optimized for size (2 bytes).
